Descripción de la actividad:
En este proyecto desarrollé una aplicación en React donde practiqué la creación de rutas, formularios, renderizado de listas, consumo de una API REST y manejo de estado con useState y useEffect.
La idea principal fue construir una pequeña aplicación que tuviera distintas páginas y que mostrara información obtenida desde una API externa.

Las rutas que implementé fueron:

/ → Página de inicio

/todos → Página donde muestro usuarios traídos desde una API

/registro → Formulario simple para registrar nombre y correo
